UMLClass «Tipo»
conacyt :: Modelo de Dominio :: Modalidades :: «Tipo»Modalidad

Description

La modalidad es el tipo de proyecto que se desea presentar y puede ser de tres tipos:

A) Individuales

B) Grupo

C) Sinergias
